# Release Notes

## Asset Management

- Assets in your library are now versioned, paving the way for stronger video, audio, and image editing capabilities.
- Introduced new browser filters for imported and unused media
- Improved filter user interface, cleaner design, invertible facets, empty options are hidden
- Audited and improved deletion behavior

## LLM Providers

- Connect OpenAI, Anthropic, Google, and xAI to Stimma directly with your own API keys, or route through OpenRouter, Together, or Fireworks in addition to Local LLMs
- Models are consistently branded and ordered across pickers

## Onboarding & Home

- New first-run setup wizard, followed by a short tour of the sidebar.
- Redesigned Home and Project screens 

## Tools

- Inpaint masks are now preserved across tool hops
- Remixes preserve the tool instance they came from.
- Added Trash and remix quick actions on finished images on tool pages
- Improved stage mode on the tool screen: seamless video looping, audio support, drag support, and context menu
- Slideshow control bar defaults to vertical so that it doesn't collide with the video scrubber.
- Improved prompt enhancement for video tools, better at preserving audio and dialogue

## Stimma Account

- Pay-as-you-go billing introduced for Chat Models and generation tools
- Cloud LLM metering audited for accuracy across all providers

## Flows

- Flows new use the tool screen's approach to adding input media
- Dragdrop on a flow on the sidebar now distinguishes between providing inputs and attaching to a chat message

## Fixes & Polish

- Window position, size, and monitor are remembered between runs.
- Fixed the chat input drag overlay getting stuck 
- Closing a slideshow no longer drops your active tool
- Fixed hidden tool videos continuing to play offscreen
- Docs: new Browse and Generation Tools sections, refreshed getting-started guide, and a site-wide dark theme.
